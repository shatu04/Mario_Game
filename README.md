
# Super Mario Game (Pygame)

A simple Mario-style platformer built with **Python** and **Pygame**.  
Includes 3 levels with enemies, power-ups, coins, and moving platforms.

## Features
- 🎮 3 Levels: Easy, Medium, Hard
- 👾 Multiple enemy types (basic, fast, jumper)
- 💎 Coins & power-ups (speed boost, jump boost)
- ❤️ Lives & scoring system
- 🏆 Win screen & Game Over screen

## Controls
- Arrow Keys / WASD → Move
- Space / Up / W → Jump
- ESC → Quit
- R → Restart (on Game Over / Victory)

## Requirements
- Python 3.x
- Pygame (`pip install pygame`)

## Run the Game
```bash
python mario_game.py
